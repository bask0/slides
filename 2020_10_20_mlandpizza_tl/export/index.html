1. Terminology
feature space $\mathcal{X}$ $n$-dimensions where features live
label (aka target) space $\mathcal{Y}$ $n$-dimensions where targets live
marginal probability distribution $P(X)$ data distribution
predictive function $f(\cdot)$ the model, learned from $\{x_i, y_i\}$, $P(y \vert x)$
domain $\mathcal{D}$ $\mathcal{D} = \{ \mathcal{X}, P(X) \}$
task $\mathcal{T}$ $\mathcal{T} = \{ \mathcal{Y}, f(\cdot) \}$
source domain $\mathcal{D}_S$ accordingly: $\mathcal{X}_S$ source feature space etc.
target domain $\mathcal{D}_T$ accordingly: $\mathcal{X}_T$ target feature space etc.

2. Categorization of transfer learning techniques

$\neq$ implies space or distribution
transductive TL $\mathcal{D}_S$ $\neq$ $\mathcal{D}_T$ $\mathcal{X}_S$ $\neq$ $\mathcal{X}_T$ $P_S(X)$ $\neq$ $P_T(X)$
inductive TL $\mathcal{T}_S$ $\neq$ $\mathcal{T}_T$ $\mathcal{Y}_S$ $\neq$ $\mathcal{Y}_T$ $P_S(Y_S \vert X_S)$ $\neq$ $P_T(Y_T \vert X_T)$

  • transductive: reasoning from observed, specific (training) cases to specific (test) cases.
  • inductive: reasoning from observed training cases to general rules.

Domain adaption (DA)

$\mathcal{D}_S$ $\neq$ $\mathcal{D}_T$

D. Tuia et al. (2016) ‚ÄúDomain Adaptation for the Classification of Remote Sensing Data: An Overview of Recent Advances,‚Äù in IEEE Geoscience and Remote Sensing Magazine doi: 10.1109/MGRS.2016.2548504.

DA aims to adapt models trained on specific task to a new, related task, for which the knowledge of the initial task are sufficient, although not perfect. In DA, the feature and label space remain unchanged.

  • Domain adaption
    • Processes (including acquisition) are not stationary over space or time.
    • Different viewing geometry, sensor, coditions (e.g. soil moisture), etc.
  • Related:
    • Sample selection bias: not independently and randomly selected sampels.
    • Covariate shift (case of sample selection bias): bias depends only on features.

Domain-adversarial training

Ganin et al. (2016). Domain-adversarial training of neural networks. The Journal of Machine Learning Research, 17(1), 2096-2030.

Deep semi-supervised learning for domain adaption

  • Source: features & labels; target: features
  • Introduce auxiliary task: reconstruction of features

Objective functions for training:

  1. main task performance
  2. reconstruction of source features & target features
  3. Maximum mean discrepancy (divergence between distributions of two data sets)

Transfer of knowledge

(another vague term)

$\mathcal{T}_S$ $\neq$ $\mathcal{T}_T$

Transfer of knowledge: semi-supervised = Self-Taught Learning

  • No source labels but target labels
  • Train unsupervised model (feature extractor) on source features
  • Use representation as input for the target task model

The model already knows how to ‚Äúwalk‚Äù before labels are used.

Transfer of knowledge: supervised

  • Source labels (auxiliary task) and target labels
  • Train unsupervised model (feature extractor) on auxiliary task
  • Use representation as input for the target task model
